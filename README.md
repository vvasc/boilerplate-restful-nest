## Instalação
Instale o PostgreSQL e configure um novo DB
```bash
$ npm install
```

## Rodando

```bash
# development
$ npm run start
```

## Testes

```bash
$ npm run test
```

## Descrição

Um simples CRUD de bicicletas com usuário, login, autenticação e testes unitários
